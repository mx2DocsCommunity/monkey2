Monkey2 v2018.09

(Mx2cc v1.1.15 • Ted2go v2.13)

Welcome to monkey2, an open source, user friendly, cross platform programming language by Mark Sibly.

Some example programs can be found in the bananas directory.

Monkey2 is a crowd funded project. If you like what you see, please consider contributing to development by becoming a Patreon supporter: https://www.patreon.com/monkey2.

For monkey2 discussion and programming help, feel free to drop by the monkey2 forums: http://monkeycoder.co.nz/forums.

Please post any bug reports or feature requests to the monkey2 github repository: https://github.com/blitz-research/monkey2.
